## Development

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Setup

Navigate to the plugin directory and Install dependencies:

```bash
npm install
```

### Building

#### Development Build (with watch mode)
```bash
npm run dev
```

#### Production Build
```bash
npm run build
```

#### Build for specific environment
```bash
# Development build
node build.js

# Production build
node build.js production
```
